# FitZone Shop — Mini Projet Web

**Rana Salem** — LSIM2E

FitZone est un site e-commerce de vente de machines de sport.
Il permet de consulter les machines disponibles, s'inscrire, se connecter, ajouter des articles au panier et envoyer des messages via le formulaire de contact.

## Technologies
HTML5 · CSS3 · JavaScript · PHP · MySQL

## Pages
- `index.php` — Accueil
- `all.php` — Toutes les machines
- `auth.php` — Connexion / Inscription
- `contact.php` — Contact
- `panier.php` — Panier

## Installation
1. Copier le dossier `fitzone` dans `C:/xampp/htdocs/`
2. Importer `database/script.sql` dans phpMyAdmin
3. Ouvrir `http://localhost/fitzone/index.php`
