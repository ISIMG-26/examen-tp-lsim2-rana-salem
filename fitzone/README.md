## Membres du groupe

## Description
Site e-commerce de vente de machines de sport. 
Le site permet de consulter les machines disponibles, s'inscrire, se connecter et envoyer des messages.

## Technologies utilisées
- HTML5 sémantique
- CSS3 externe (css/style.css)
- JavaScript natif externe (js/script.js)
- PHP natif
- MySQL (via XAMPP)

## Pages
- `index.php` — Page d'accueil
- `auth.php` — Connexion / Inscription
- `all.php` — Toutes les machines
- `contact.php` — Formulaire de contact

## Installation (XAMPP)
1. Copier le dossier `fitzone` dans `C:/xampp/htdocs/`
2. Ouvrir phpMyAdmin : http://localhost/phpmyadmin
3. Créer la base en important `database/script.sql`
4. Lancer le site : http://localhost/fitzone/index.php
